## Source Priority Legend

- `P1_core`: peer-reviewed core source or older foundational/standard work.
- `P2_frontier`: recent arXiv/frontier signal, not sole support for key conclusions.
- `P3_background`: useful background but not central evidence.
- `downgraded`: weak evidence, use only with caveat.
- `excluded`: not used as evidence.

## Evidence Grade Legend

- `strong`: directly relevant, transparent protocol, strong baselines, strong claim-evidence fit.
- `medium`: useful but with some limitations.
- `weak`: auxiliary only.
- `low_confidence`: mention only with explicit caveat.
- `reject`: do not cite as support.
