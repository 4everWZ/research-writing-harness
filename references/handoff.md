# Paper Handoff

## Scope

Use this reference only when the user requests paper handoff or when a Tier A/B paper-state change has occurred.

Do not update `handoff.md` for local edits.

## Update Triggers

### Tier A: Must update handoff

- paper direction changes;
- contribution framing changes;
- method direction changes;
- baseline strategy changes;
- dataset / split / metric / protocol changes;
- workspace target venue/outlet confirmation that affects writing or evaluation framing.

### Tier B: Should update handoff

- completed literature collection round;
- completed major section draft;
- important literature-driven idea refinement;
- discovery of strong similar work;
- discovery of major evidence gap;
- major citation audit finding.

### Tier C: Do not update handoff

- local wording edits;
- single citation addition;
- BibTeX formatting fix;
- table formatting;
- typo fixes.

## Handoff Contents

Record:

- current paper goal;
- target venue/outlet if known;
- current writing status;
- literature collected;
- strong baselines;
- weak or risky sources;
- claims currently allowed;
- claims not yet allowed;
- sections drafted;
- missing evidence;
- required experiments before claims;
- next recommended actions;
- do-not-forget notes.

## Output Format

When reporting handoff work, include:

1. Why handoff was updated;
2. Tier classification;
3. Key state changes;
4. Files updated.
